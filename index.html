CSS Box Model Quiz

Master padding, borders, and margins usingcontent-box calculations and coding techniques.

Part 1: Calculations & Concepts

1 Total Width Calculation

Calculate the total horizontal space occupied by this element:

.box {
  width: 300px;
  padding: 20px;
  border: 5px solid black;
  margin: 15px;
 }
View Solution

Total Width = 300px+(20px × 2) + (5px x 2) + (15px x 2)

300+40+10+30= 380px

Result:380px

2 Padding Shorthand

If an element is styled with padding: 10px 25px;, what are the specific values for each side?

View Solution
  • Top & Bottom:10px
  • Left & Right:25px

3 Vertical Margin Collapse

Box A has margin-bottom: 40px;.(immediately below) has margin-top: 25px;. What is the actual vertical gap between them?

View Solution

Due to margin collapse, the gap is equal to the maximum of the two adjacent margins:

Gap = max(40px, 25px) = 40px

4 Total Height Calculation

Calculate the total vertical space occupied by this element:

.box {
        ol>li*9
  height: 150px;
  padding-top: 10px;
  padding-bottom: 30px;
  border-top: 2px solid red;
  border-bottom: 0px;
  margin-top: 20px;
  margin-bottom: 20px;
 }
View Solution

Total Height = 150px+10px+30px+2px+0px+20px+20px

Result:232px

5 Border Shorthand Identification

If an element is styled with 20px : border-width: 5px 10px 15px 20px;?

View Solution

Order: Top (5px), Right (10px), Bottom (15px), Left (20px).

6 Negative Margins

If an element has margin-left: -30px;, in which direction does the element move?

View Solution

The element is pulled 30px to the left (potentially overlapping the container or previous siblings).

7 Padding vs Margin

Which layer is located inside the border and shows the background color?

View Solution

Padding.

8 Centering

What property value pair is used for centering a block element horizontally?

View Solution

margin: 0 auto;

9 Percentage Padding

Padding percentages are relative to what property of the parent?

View Solution

The Widthof the parent container.

10 Layer Order

List the box model parts from innermost to outermost.

View Solution

Content → Padding → Border → Margin.
